Weather Dashboard

This is a simple Weather Dashboard project built using HTML, CSS and JavaScript.

 Description
The user can enter a city name and get real-time weather details using a public Weather API.

Features
- Search weather by city name
- Shows Temperature
- Shows Wind Speed
- Shows Humidity
- Shows Pressure
- Shows Current Time

Technologies Used
- HTML
- CSS
- JavaScript
- Weather API

How to Run
1. Download the project
2. Open index.html in browser
3. Enter city name
4. Click Get Weather
